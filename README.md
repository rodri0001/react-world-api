## Clone the repository   

```bash
git clone https://github.com/rodri0001/react-world-api.git
```

Navigate to the project directory:

```bash
cd react-world-api
```

Install dependencies:

```bash
npm install
```

## Usage

Start the development server: 

```bash
npm run dev
```
